# ts-atcoder

## Description
[AtCoder](https://atcoder.jp/home)の問題を解いたコードを管理するリポジトリです。

## Usage
### Install
```bash
$ npm install -g atcoder-cli
$ pip3 install online-judge-tools
```

### Login
```bash
$ acc login
$ oj login https://atcoder.jp 
```

### New Contest
```bash
$ acc new abc001
```

### Submit
```bash
$ acc submit main.ts -- -l 5058
```
